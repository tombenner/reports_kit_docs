When using a `data_method` that returns data with one dimension (i.e. a single series of data), the `data_method` can return the data in one of two formats:

### Hash

```ruby
{ 'Bernice' => 191, 'Carlee' => 160, 'Edison' => 60 }
```

### Array

```ruby
[['Bernice', 191], ['Carlee', 160], ['Edison', 60]]
```
